# citycountryhebrew-8964

A digital version of the classic Israeli word game 'Eretz Ir' (Country City) played in Hebrew, where users compete to name items starting with a chosen letter across categories like cities, countries, animals, and more, with real-time multiplayer support and scoring.

## Tech Stack
Vanilla HTML/CSS/JS + Supabase

## Setup
Run `docker-compose up` to start the infrastructure.
